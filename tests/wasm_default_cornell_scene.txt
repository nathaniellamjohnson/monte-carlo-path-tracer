### WASM Default Cornell Scene (LLM-Friendly Template)
###
### This file is intentionally verbose so an LLM can generate new scenes.
###
### Supported top-level directives in this renderer:
### - size W H
### - recursion_depth_limit N
### - enable_shadows 0|1
### - enable_caustics 0|1
### - photon_mapping_params photons_per_light max_photons_gathered gather_radius
### - spp samples_per_pixel
### - color name r g b
### - phong_shader name ambient_color diffuse_color specular_color specular_power
### - flat_shader name color is_emissive(0|1)
### - reflective_shader name base_shader_name reflectivity
### - glass_shader name tint_color ior
### - plane point_x point_y point_z normal_x normal_y normal_z shader_name
### - sphere center_x center_y center_z radius shader_name
### - point_light x y z color_name brightness
### - ambient_light color_name intensity
### - camera pos_x pos_y pos_z look_x look_y look_z up_x up_y up_z fov_degrees
###
### Notes for scene generation:
### - Keep geometry inside Cornell walls roughly within x in [-1,1], y in [-1,1], z in [-1,4].
### - Caustic photons are emitted from point_light in this implementation.
### - Emissive flat shaders help path tracing converge faster at low spp.

### Global Settings ###
size 512 384
recursion_depth_limit 6
enable_shadows 1
enable_caustics 1
# Interactive defaults for browser speed; raise these for final quality.
photon_mapping_params 60000 220 0.05  # photons_per_light max_photons_gathered gather_radius
spp 16

### Color Definitions ###
color red 0.75 0.1 0.1
color green 0.1 0.75 0.1
color white 0.78 0.78 0.78
color blue 0.1 0.1 0.75
color yellow 1.0 0.9176 0.0
color light_color 1.3 1.3 1.3
color diffuse_gray 0.62 0.62 0.62
color glass_tint 0.9 0.9 1.0

### Shaders ###
# Cornell wall materials
phong_shader red_wall red red white 8
phong_shader green_wall green green white 8
phong_shader white_wall white white white 8
phong_shader blue_wall blue blue white 8
phong_shader yellow_wall yellow yellow white 8

# One true diffuse ball material
phong_shader diffuse_ball diffuse_gray diffuse_gray white 4

# Mirror and glass materials
reflective_shader mirror_ball white_wall 0.85
glass_shader glass_ball glass_tint 1.5

# Emissive top plane material (acts as an area light for path tracing)
flat_shader top_emissive light_color 1

### Geometry: Cornell Box ###
plane -1 0 0 1 0 0 red_wall
plane 1 0 0 -1 0 0 green_wall
plane 0 -1 0 0 1 0 white_wall
plane 0 1 0 0 -1 0 top_emissive
plane 0 0 -2.5 0 0 1 blue_wall
plane 0 0 4 0 0 -1 yellow_wall

### Geometry: Inner Objects ###
# left = mirror, center = diffuse, right = glass
sphere -0.52 -0.62 0.02 0.28 mirror_ball
sphere 0.00 -0.66 0.15 0.25 diffuse_ball
sphere 0.52 -0.62 0.00 0.28 glass_ball

### Lighting ###
# Keep one point light for caustic photon emission in this codebase.
point_light 0.15 0.88 -0.12 light_color 3.5
ambient_light white 0.01

### Camera ###
camera 0 0 3.5 0 -0.05 0 0 1 0 45
